# Protecting PII Article Media

Publication assets for `docs/blog/protecting-pii-in-elixir.md`. Every visible
value is synthetic.

| Asset | Intended use |
| --- | --- |
| `obscura-workbench-fast-detection.jpg` | Primary article image and wide social preview |
| `obscura-workbench-fast-anonymized.jpg` | Alternate still for launch posts about redaction |
| `obscura-workbench-vault-llm.jpg` | Article image and social still for the LLM workflow |
| `obscura-pii-boundary-workflow.gif` | Inline animation where autoplay video is unavailable |
| `obscura-pii-boundary-workflow.mp4` | Ten-second silent social and article video |

The still images and video are `1440x900`. The GIF is reduced to `960x600`,
8 frames per second, and 128 colors to keep it near 1 MB.

Suggested video description:

> Obscura detects structured PII, replaces it before an application handoff,
> and pseudonymizes an email before a simulated LLM provider call.
